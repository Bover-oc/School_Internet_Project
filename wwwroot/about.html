The Website


This website was built as part of a school internet project. It uses HTML, CSS and JavaScript for the frontend, as well as c# for the backend.

Part What it does
HTML pages Provide the content for Home, About, Photography, Paintings & Drawings and Video art & Animations.
CSS file Keeps the layout clean and consistent without a complex design.
JavaScript file Handles a few basic tasks such as the dynamic features and one API call.
C# file Serves the pages and handles the new comments in the comment section below.
